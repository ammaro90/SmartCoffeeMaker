Project Title: Smart Coffee Maker
Authors: Amar Awad Ismail, Heng Sing Song
Purpose: This project is for university assignment at FKE, UTeM.
Date: 18/10/2015

Project Details:
The smart coffee maker project is about modifiying a small coffee maker to control its operation over the internet.

This project is an application of Iot "Internet of Things" where we will connnect the coffee maker to an arduino uno. This arduino is connected through an ethernet cable to the internet.

A web application will be created as a user interface for the coffee maker.
